## ğŸ”§ Troubleshooting

### Issue: "Did not find xcodeproj from ios"

**This is expected!** You need to generate the iOS project first:
```bash
flutter create --org com.totv --platforms=ios .
```

### Issue: "CocoaPods not installed"

**Solution:**
```bash
sudo gem install cocoapods
pod setup
```

### Issue: "google-services.json not found"

**Solution:** Download from Firebase Console and place in `android/app/`

### Issue: "No such module 'Firebase'"

**Solution:**
```bash
cd ios
pod install --repo-update
cd ..
flutter clean
flutter pub get
```
